# Public Pilot Pack

This document summarizes what is ready to share with pilot users.

## Public Links

- Landing page: `https://daltieri86.github.io/DAORoute-landing/`
- Public MCP pilot repository: this repository
- Contact: `softwaretamrsv@gmail.com`

## Pilot User Receives

- MCP endpoint URL;
- API key;
- configuration example;
- sample requests;
- redacted sample responses;
- onboarding guide;
- response field guide;
- safety and non-custodial boundaries;
- structured feedback template;
- feedback instructions.

## Pilot User Does Not Receive

- private engine source;
- raw datasets;
- model files;
- ingestion credentials;
- database access;
- execution custody;
- transaction signing rights.

## Good Pilot Fit

- AI agent builders;
- DeFi automation builders;
- wallet teams testing agent workflows;
- DAO treasury automation researchers;
- MCP client developers;
- protocol teams exploring non-custodial routing evidence.

## Poor Pilot Fit

- users seeking guaranteed yield;
- users asking for trading signals;
- users expecting custody or auto-execution;
- users requesting raw database exports;
- users asking to clone or self-host the proprietary engine.
