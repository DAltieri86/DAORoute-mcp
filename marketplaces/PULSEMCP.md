## Description

```text
DaoRoute is currently available as a free controlled pilot. It exposes a hosted Streamable HTTP MCP server for approved AI-agent and DeFi automation builders.

The server packages decision evidence for agents: aggregate market coverage, one-pool evidence, live protocol security status, and stablecoin allocation decision packets. Responses are designed for policy review, simulation, and user approval, not automatic custody or transaction broadcasting.

DaoRoute does not custody funds, sign transactions, broadcast transactions, guarantee returns, provide investment advice, expose raw datasets, or distribute private model internals.
```
